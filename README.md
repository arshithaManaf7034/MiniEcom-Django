# Mini E-Commerce Platform (Django)

## Features
- User authentication (Login/Register)
- Product listing
- Cart system
- Checkout system
- Order & OrderItems
- Admin panel

## Tech Stack
- Django
- SQLite
- Bootstrap

## How to Run
git clone <repo>
cd MiniEcom
python3 -m venv venv
source venv/bin/activate
pip install django
python manage.py runserver
